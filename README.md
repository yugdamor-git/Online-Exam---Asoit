# Description

## Features

### STUDENTS :- 
1. It can handle any number of users.(sky is the limit) 

2. The application will never ever slow down when lots of users use it. 

3.It will give same access time regardless of number of users. 

4. Imagine you're giving online exam and your Internet shuts down. - Don't worry it won't affect this application.  The flow will remain the same. Even without Internet you can select answer. And it will show you the result when you submit or the time is over. And it will update all the data to server as soon as you turn the Internet on. 

5. It will send you notifications before exam. 

### Faculty :- 

1. They can upload questions and answers in form of Excel file. There is predefined excel file in which they have to fill all the details. Thn that can directly uploaded to server. 

2. Faculty have full control over any exam. They can show or hide test any point of time. 

3. Faculty have option to upload test in unpublished tab. So they can check whether everything is right or not.and at that time it won't be visible to students. 

4.faculty doesn't have to generate results.  It will done automatically 

5. Faculty can see real time update of exam - How many students attempting exam - Which student has completed how many questions. -Who didn't enroll in exam And there is call button in that tab so they can directly call them. 

## LOGIN:- 

1. OTP will be send on your number for login.
